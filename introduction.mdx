## What is ZeroTier?

ZeroTier is a **smart programmable Ethernet switch for the planet**. It combines two layers:

- **VL1** — A cryptographically addressed peer-to-peer network. Every device gets a unique 10-digit ZeroTier address derived from its public key. Peers connect directly to each other across any underlying network topology, with automatic NAT traversal.
- **VL2** — An Ethernet emulation layer similar to VXLAN, built on top of VL1. It provides virtual switches, fine-grained access control rules, IP address management, and multicast.

All ZeroTier traffic is **end-to-end encrypted** using secret keys that only you control. Most traffic flows peer-to-peer without touching ZeroTier's servers. A free relay service handles fallback cases where direct peer connections cannot be established.

## Getting started

<Steps>
  <Step title="Install ZeroTier">
    Download and install the ZeroTier One service on your device. See the [installation guide](/installation) for platform-specific instructions.
  </Step>
  <Step title="Create or join a network">
    Create a free network at [my.zerotier.com](https://my.zerotier.com) or join an existing one with its 16-digit network ID.
  </Step>
  <Step title="Authorize your device">
    Your device appears in the network's member list. Authorize it (for private networks) and it receives an IP address.
  </Step>
  <Step title="Connect">
    All authorized devices on the network can communicate as if on the same LAN — regardless of physical location.
  </Step>
</Steps>

<Tip>
  For a hands-on walkthrough, see the [Quick Start guide](/quickstart) which takes you from installation to a working connection in minutes.
</Tip>
